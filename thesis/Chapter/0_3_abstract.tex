\chapter*{ABSTRACT}
\addcontentsline{toc}{chapter}{ABSTRACT}

The freelance economy has grown significantly in recent years, but traditional centralized platforms suffer from high fees, payment delays, and trust issues. This thesis presents a decentralized freelance marketplace built on the Solana blockchain that addresses these challenges through smart contracts, trustless escrow, and transparent reputation systems.

The platform implements a hybrid dual-flow hiring mechanism that supports both traditional job-first approaches (where clients post jobs and receive bids) and innovative freelancer-first approaches (where clients browse freelancer profiles and initiate direct hiring). The system features automatic role detection using blockchain data analysis, eliminating manual user categorization and improving the overall user experience.

Key technical contributions include: (1) a SOLID-compliant architecture ensuring maintainability and extensibility, (2) real-time notification system with auto-polling for job events, (3) advanced filtering and search capabilities for jobs and freelancers, (4) hybrid role detection system achieving 95\% accuracy, and (5) escrow-based payment mechanism with dispute resolution.

The platform is built using Next.js and TypeScript for the frontend, Anchor framework for Solana smart contracts, and implements various design patterns including Repository, Strategy, and Dependency Injection. The system achieves a 5x increase in freelancer discoverability compared to reputation-only approaches.

Evaluation results demonstrate that the platform successfully addresses the limitations of centralized marketplaces while maintaining high performance (up to 65,000 TPS on Solana) and low transaction costs. The decentralized architecture ensures transparency, reduces intermediary fees, and provides users with complete control over their funds and data.

\textbf{Keywords:} Blockchain, Solana, Decentralized Application, Freelance Marketplace, Smart Contracts, Escrow, Reputation System, SOLID Architecture.
